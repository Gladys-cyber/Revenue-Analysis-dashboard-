# Revenue Analysis Dashboard
Data analysis project using Excel/Power BI to uncover revenue trends and insights.
## 🛠️Tools used
-Excel
-Power Bi 
## Objective
To analyse revenue data and identify trends that support business decision-making.
## Key Insights
- Identified upward trend in enrollment over time
- Revenue showed consistent growth across key periods
- Top-performing categories contributed the highest share of revenue
